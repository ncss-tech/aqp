# Rebuild a SoilProfileCollection object

Rebuild a SoilProfileCollection object

## Usage

``` r
rebuildSPC(x)
```

## Arguments

- x:

  a `SoilProfileCollection` object

## Value

A valid `SoilProfileCollection` object.

A valid `SoilProfileCollection` object.

## Details

Attempt rebuilding a `SoilProfileCollection` object by splitting into
components and re-assembling. Likely only used to fix outdated
`SoilProfileCollection` objects that are missing slots.
