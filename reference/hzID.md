# Set horizon IDs

Set vector containing horizon IDs

Get vector containing horizon IDs

## Usage

``` r
# S4 method for class 'SoilProfileCollection'
hzID(object) <- value

# S4 method for class 'SoilProfileCollection'
hzID(object)
```

## Arguments

- object:

  a SoilProfileCollection

- value:

  a unique vector of equal length to number of horizons `nrow(object)`
