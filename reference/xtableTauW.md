# Format a LaTeX table with results

Format a LaTeX table with results

## Usage

``` r
xtableTauW(result.tau, file.name = "tau_results_table.tex")
```

## Arguments

- result.tau:

  results returned by `tauW`

- file.name:

  name of file to write output TeX file; Default:
  `file.name="tau_results_table.tex"`
