# Convert cross-tabulation to adjacency matrix.

Convert a cross-tabulation of e.g. original horizon designations vs.
generalized horizon labels to adjacency matrix form.

## Usage

``` r
genhzTableToAdjMat(tab)
```

## Arguments

- tab:

  table, cross-tabulation of original and generalized horizon labels
  e.g. `table(original, genhz)`

## Value

matrix of numeric values
